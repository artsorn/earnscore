ช่วยปรับปรุง  โดยต้องการดึงข้อมูลทั้งหมดของ football, basketball ของเว็บ https://m.aiscore.com/ สำหรับบอล และ https://m.aiscore.com/basketball แล้วเข้าไปดึงข้อมูลทั้งหมด (ตอนนี้ดึงไม่หมด)  เท่าที่จะสามารถดึงได้ (ไม่ต้องเอาเมนู chat) บันทึกลง sqlite โดยสามารถแบ่ง cli เป็น 2 session คือ football, basketball ที่หน้า Live ค้างไว้ หาก event ใดๆ เปลี่ยน ก็บันทึกลง db ตามถามทีมใหม่ แสดงเข้ามาที่หน้า  Live ที่เปิดค้างไว้ ก็ให้เข้าไปดึงข้อมูล ทุกอย่าง (ยกเว้น chat)  ถ้าทีมแข่งเสร็จ ก็เปลี่ยนสถานะตาม ว่าแข่งเสร็จแล้ว
และช่วยช่วยหน้าเว็บสำหรับแสดงข้อมูล (แสดงสกอร์ live football, live basketball เหมือนต้นฉบับ และคลิ้กแสดงรายละเอียด เหมือนต้นฉบับ) สำหรับแสดงข้อมูลได้ (worker cloudflare) โดยให้สามารถตั้งค่าได้ว่า จะ อัพเดทข้อมูลทั้งหมด ลง d1 ของ cloudflare ในทุกๆ กี่นาที
ใช้ภาษา rust เป็นหลัก และคุณสามารถใช้  mcp ของ chrome-devtools สำหรับดึงข้อมูล ดูโครงสร้างต่างๆ และใช้ทำงานต่างๆได้
